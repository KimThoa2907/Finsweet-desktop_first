# Finsweet Landing Page

## Description
This project is a responsive landing page built based on the Finsweet design.  
The website was developed from scratch to practice front-end development and responsive layout implementation.

## Technologies Used
- HTML
- Bootstrap 5
- CSS (SASS)
- JavaScript

## Live Demo
https://kimthoa2907.github.io/Finsweet-desktop_first/

## Features
- Responsive design for desktop, tablet, and mobile devices
- Clean UI layout following the original design
- Modern CSS styling using SASS
